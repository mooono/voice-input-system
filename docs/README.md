# Voice Input System (Windows + Azure)

## Overview
This project implements a voice input user interface similar to TypePress.

Users can:
- Press a hotkey
- Speak into the microphone
- Convert speech to text
- Optionally refine the text
- Insert it into the active application

## Tech Stack
- Python (prototype)
- Azure Speech-to-Text
- (Optional) Azure OpenAI
- Windows environment

## Key Features
- Real-time speech recognition
- Low-latency transcription
- Secure communication
- Extensible architecture

## Constraints
- Avoid public STT services
- Use private cloud or secure configuration
