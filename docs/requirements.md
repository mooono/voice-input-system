# Requirements

## Functional Requirements

- Start/stop recording via hotkey
- Real-time speech recognition
- Display partial transcription
- Display final transcription
- Insert text into active window
- Optional text refinement

## Non-Functional Requirements

### Performance
- Latency < 1 second for partial results
- Final result within 2–4 seconds

### Security
- No persistent storage of audio
- Encrypted communication
- Secure key management

### Usability
- Simple hotkey operation
- Minimal UI
- Clear feedback to user

## Environment

- Windows 10 or later
- Python 3.10+
